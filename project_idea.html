Project Idea

Overview

I am unsure about the scope the project can take within the limits of this course, so I will outline my idea in levels. The overall goal is to create a system inspired by Skylanders or Nintendo’s Amiibos, but designed for Dungeons & Dragons (DnD) characters.

Level 1

Similar to Skylanders or Amiibos, I want to create figures of DnD characters that store their character’s information within their base. Then, I would like to create an external box-like device, with a scanner on top, which continuously reads and writes onto the character, to display the character sheet.

Level 2

I would like additional inputs, such as a calculator that contains numbers from 0 to 9, a clear and remove key, and a plus and minus key. This would allow you to write a number to add or remove from your character's health.

Level 3

Additionally, I would like to add buttons for each DnD die, so that you can easily roll, keeping the last roll displayed on screen. These would be buttons for a d4, d6, d8, d10, d12, d20, and d100, ordered by importance (so the d20 would be first).

An alternative design would be a dial that has different intervals mapped to the different dice. Then, if possible, I would like to light up the dice display of the currently selected die.

Level 4

Next, above the display showing initiative and modifiers, I want to add a slider to select what modifier you want to roll for. Next to the display, on the left, I would include a switch to toggle between normal rolls and saving throws.

Level 5

(To be determined) — At this stage, I would refine or expand on the interaction design based on what’s feasible within the course. Possible extensions could include LED feedback, modular character stands, or integration with a companion web app for digital tracking.
